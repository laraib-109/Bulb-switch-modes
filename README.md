Bulb On/Off Project 💡

A simple web project to simulate turning a bulb **ON** and **OFF** using JavaScript. This project demonstrates basic DOM manipulation and event handling in a fun and interactive way.

---

## Features

- Turn the bulb **ON** or **OFF** with a button click.
- Visual effect to show the bulb glowing when turned on.
- Fully responsive layout for mobile and desktop screens.
- Easy-to-understand HTML, CSS, and JavaScript code.
